A rock solid WordPress contact form plugin that focuses on spam prevention, security, and mail deliverability.

== Description ==

Rock Solid Contact Form is the result of years of experience managing a network of WordPress websites. Built with security and reliability in mind, this plugin eliminates unnecessary complexity.

- No form building requiredâ€”just use `[rock-solid-contact]` and it works.
- Spam prevention features without the need for CAPTCHA.
- Secure file attachments with configurable format and size limits.
- Email deliverability enhancements to ensure messages arrive as expected.
- Easy-to-use settings pages for Form, Attachments, Spam, and Email configurations.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the Settings page to configure fields, email, and spam settings.
4. Place the shortcode `[rock-solid-contact]` into a page, post, or widget.
